Warm up
Write a function that returns a 10 X 10 multiplication table (2D array).

For example:

multTable();
// Output:
//  [
//    [1,2,3,4,5...],
//    [2,4,6,8,10...],
//    [3,6,9,12,15...],
//    ...
//  ]
BONUS:
Allow for the function to take a parameter n and return a table with n * n cells.

multTable(n);
// Output:
//  [
//    [1,2,3,4,5...],
//    [2,4,6,8,10...],
//    [3,6,9,12,15...],
//    [n, n * 2, n *3, etc...]
//  ]
